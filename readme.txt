Add a customizable slidebar with icons and links to the side of your website.

== Description ==

MioSlide Bar adds a beautiful, customizable slidebar to the right or left side of your website. Perfect for social media links, contact buttons, or important CTAs.

= Features =

* Customizable icons (up to 5)
* Choose left or right position
* Tooltips on hover
* Toggle show/hide functionality
* Responsive design
* Color customization
* Works with Dashicons and Font Awesome

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mio-slide-bar` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin settings under 'MioSlide Bar' in the WordPress admin menu.

== Frequently Asked Questions ==

= Can I add more than 5 icons? =
Yes, you can modify the code to add more icons by editing the plugin files.

= Does it work with Font Awesome? =
Yes, if you have Font Awesome installed on your website, you can use Font Awesome classes for the icons.
